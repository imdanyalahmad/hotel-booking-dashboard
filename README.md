# Hotel Bookings Analysis - Excel Report

## Project Overview
This Report Analyzes bookings data to understand patterns of booking and cancellation of Customers and their behaviour.
Project Objective was to convert Raw Data into meaningful insights.
The Report is created using Microsoft Excel.
# Dataset
The Dataset contains the Record of Bookings.
## Dashboard Features
### Summary
Overview of Hotel Bookings Performance.

Key Metrics:
- Total Bookings
- Total Cancellations
- City vs Resort(Bookings)
- City vs Resort(Cancellations)

Insights include:
- Bookings & Cancellations with respect to Guest type
- Bookings & Cancellations with respect to Room type
- Months with highest bookings and cancellations
### Cancellation Insights
- City Hotels have more bookings and cancellations then resorts
- Couples have highest bookings and cancellations
- Certain months shows high cancellations
- Portugal has highest number of cancellations
